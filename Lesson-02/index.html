Урок 02 - Основные операторы JavaScript

  • Задание 01

    Почему код дает именно такие результаты?

    var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2 - сначала происходит инкремент переменной 'a', затем ее значение присваивается переменной 'c'
d = b++; alert(d);           // 1 - сначала переменной 'c' присваивается значение из 'b', затем происходит инкремент 'b', результат присваивается в 'd'
c = (2+ ++a); alert(c);      // 5 - сначала происходит инкремент 'a' затем ее значение складывается с '2', результат присваивается в 'c'
d = (2+ b++); alert(d);      // 4 - сначала '2' складывается с значением из 'b' (это помещается в 'd'), значение в 'b' инкрементируется
alert(a);                    // 3 - значение переменной 'a' было дважды инкрементировано в предыдущих инструкциях
alert(b);                    // 3 - значение переменной 'b' было дважды инкрементировано в предыдущих инструкциях
