Например, для числа 245 надо получить следующий объект: {‘единицы’: 5, ‘десятки’: 4, ‘сотни’: 2}. Если число превышает 999, необходимо выдать соответствующее сообщение с помощью console.log и вернуть пустой объект.

    Был создан отдельный клас Product для хранения информации о товаре, с полями name и price, хранящими имя и цену товара. И так же класс Item для представляющий позицию конкретного товара в корзине (или к примеру в каталоге), который содержит поля product и quanity содержащими ссылку на объект класса Product и количество товара для этой позиции.

    Для представления корзины был создан класс Basket, содержащий приватное поле products, для хранения объектов класса Item

    У класса корзины есть два основных метода:

    • countPrice - возвращает цену всех товаров в корзине;

    • addProduct - добавляет товар в хранилище корзины (с проверкой входного параметра на принадлежность к классу Product)

    У классов Product и Item есть методы для чтения приватных полей, а так же для подсчета полной стоимости позиции (класс Item)
