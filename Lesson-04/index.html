Например, для числа 245 надо получить следующий объект: {‘единицы’: 5, ‘десятки’: 4, ‘сотни’: 2}. Если число превышает 999, необходимо выдать соответствующее сообщение с помощью console.log и вернуть пустой объект.

  • Задание 02

    Продолжить работу с интернет-магазином. Задачи:

    • В прошлом домашнем задании была реализована корзина на базе массивов. Какими объектами можно заменить их элементы?

    • Реализовать такие объекты.

    • Перенести функционал подсчета корзины на объектно-ориентированную базу.


    Был создан отдельный клас Product для хранения информации о товаре, с полями name и price, хранящими имя и цену товара.

    Для представления корзины был создан класс Basket, содержащий приватное поле products, для хранения объектов класса Product

    У класса корзины есть два основных метода:

    • countPrice - возвращает цену всех товаров в корзине;

    • addProduct - добавляет товар в хранилище корзины (с проверкой входного параметра на принадлежность к классу Product)


    Содержимое корзины
